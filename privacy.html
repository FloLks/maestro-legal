Privacy Policy — MAESTRO HATHOR

Last updated: April 18, 2026

1. Who We Are

MAESTRO HATHOR is a personal application developed and operated solely by Florent Liekens, owner of the Facebook Page "Eurocombles". This application is for personal use only.

2. What Data Is Collected

The application accesses the following data from Facebook, strictly limited to the Page "Eurocombles":

  • Page information (name, ID)
  • Post engagement metrics (likes, comments, shares, reach)
  • Comments on posts
  • Draft posts created by the Page owner

No data from other Facebook users, other Pages, or third parties is collected.

3. How Data Is Used

All data accessed is used exclusively by the Page owner (Florent Liekens) to:

  • Draft and publish content to the Eurocombles Page
  • Analyze the performance of his own posts
  • Respond to comments from his audience

Data is never shared, sold, or transmitted to any third party.

4. Data Storage

Data is stored on a private server operated by Florent Liekens, protected by standard security measures (encrypted storage, restricted access). Data is retained only as long as necessary for the Page owner's editorial analysis.

5. Data Deletion

Users (i.e., the Page owner) may request data deletion at any time by contacting salynlol@hotmail.fr. Upon request, all stored data will be permanently deleted within 30 days.

6. Third Parties

This application uses the Facebook Graph API and the Anthropic Claude API. No user data is shared with these services beyond what is strictly necessary to operate the application (API requests with the Page owner's authentication).

7. Changes To This Policy

This policy may be updated. Changes will be posted at this URL with an updated date.
